Background

Chemical kinetics is the study of how fast reactions occur and what factors influence their rates. One important goal is to determine a rate law, which shows how rate depends on reactant concentrations.

In this experiment, iodide ions react with iron(III) ions:

$2I^- (aq) + 2Fe^{3+} (aq) \rightarrow I_2 (aq) + 2Fe^{2+} (aq)$

As the reaction proceeds, the solution changes from pale yellow to a darker brown/orange color due to the formation of iodine ($I_2$). This color change is monitored using a Vernier Go Direct SpectroVis Plus spectrophotometer.

You will measure how fast absorbance increases over time to determine:

  • Reaction order for each reactant
  • The rate law expression
  • The rate constant, $k$

In addition, this experiment relies on Beer’s Law:

$A = \varepsilon \ell c$

where absorbance ($A$) is directly proportional to concentration ($c$). By first creating a calibration curve of absorbance vs. concentration, you will be able to convert absorbance data into concentration and determine reaction rates in units of molarity per second.

Materials

  • LabQuest 3
  • Vernier Go Direct SpectroVis Plus spectrophotometer
  • 5 × 50–100 mL beakers
  • 3 × 25 mL graduated cylinders
  • Plastic transfer pipets
  • Cuvettes
  • 0.020 M KI solution (~100 mL)
  • 0.020 M FeCl₃ solution (~100 mL)
  • Distilled water (~60 mL)

Procedure

Part 1: Setup

  1. Put on goggles.
  2. Obtain all materials.
  3. Connect the SpectroVis Plus to the LabQuest 3 using USB.
  4. Tap File → New.

Part 2: Preparing an Absorbance vs. Concentration Curve.

Part 3: Calibration and Wavelength Selection

  1. Fill a cuvette ¾ full with distilled water (blank).
  2. Select Sensors → Calibrate → Spectrometer and complete calibration.
  3. Determine $\lambda_{max}$:
    • Insert FeCl₃ solution
    • Tap the Collect button
    • Identify the peak wavelength
    • Record $\lambda_{max}$ below
  4. Set the wavelength:
    • Switch to absorbance mode
    • Enter $\lambda_{max}$ so only that wavelength is measured
  5. Set time-based collection:
    • Rate: 1 sample/second
    • Duration: 200 seconds

Part 4: Collecting Data

  1. Measure KI, FeCl₃, and water.
  2. Add water (if used) to KI first.
  3. Prepare the cuvette and LabQuest.
  4. Add FeCl₃ to KI and swirl.
  5. Immediately:
    • Fill cuvette (~¾ full)
    • Insert into spectrophotometer
    • Tap Collect to begin data
  6. Allow data collection to finish.

Part 5: Determining Initial Rate

  1. Select an early linear region of the graph by dragging across it.
  2. Tap Analyze → Curve Fit
    Select Linear
  3. The slope of the line is the initial rate.
  4. Use the same time interval for all trials.
