Numerical Data Input (this is an input students put their measurements into) This will create an "input" cell for students to input a number for data collection. The id and name need to be the SAME and unique (not the same as another id). Calculated Numerical Input and Feedback (this is an input students put the results of their calculations into)
The id and name need to be the SAME and unique. The formula is how the program will check to see if the students got the answer correct. If you want it to pull numbers from another input, then put the id of that input inside of ${}. You can also use regular numbers in the formula. help is what will be shown to students when they get a calculation wrong. There is a default range="5%" that will count answers as correct if they are within 5% of the correct value calculated by the website. This could be changed by inserting something like range="5%" into the input tag. Each Calculated Numerical Input should be followed by a feedback div. The id and name need to match the Calculated Numerical Input id and name but end in "FB". You can also have TEXT input and feedback. Just change the type to "text", and the "formula" to whatever text you want to be correct. With text input you can also set multiple correct answers using "||" to mean "or". E.g. formula="Na+||Na1+||Na+1". Open Ended Input Any open ended questions will need a textarea for input. They only need an id and name that are the SAME and unique. Drop down menus
The id and name need to be the SAME and unique. Notice that in this case the "formula" is just text of the correct answer. Chemical Symbols (mass number and/or charge)
The id and name need to be the SAME and unique. The formula is the unformatted text of the chemical symbol You must include a div with the same id, but with "DIV" on the end. Notice that the type is "text", and the class includes "symbol". LaTeX $\frac{1}{3}$ LaTeX needs to be surrounded by dollar signs.
