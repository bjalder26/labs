Directions:

Develop your understanding: Explore the simulation to develop your own ideas about "isotopes". Explain your understanding using your own words to show you can:

Question 14: Define “isotope” using the following words in your definition: mass number, atomic number, number of protons, neutrons and electrons.

An isotope is an atom of an element that has the same atomic number, that is the same number of protons, but a different number of neutrons and therefore a different mass number.

Question 15: Compare and contrast: an element, an atom, and an isotope.

An element is a pure substance containing only atoms with the same number of protons. An atom is the smallest unit of an element that still retains the properties of an element. Isotopes are atoms of a given element that contain different numbers of neutrons and therefore have different masses.

Question 16: Given the number of protons, neutrons and electrons, explain how you would find the mass and name of an isotope.

If I was given the number of protons, I could identify the name of the element by looking at that atomic number on the periodic table. If the number of protons and electrons are the same, I could determine the isotope is neutral. If there were more electrons than protons, I could determine the negative charge. If there were less electrons than protons, I could determine the positive charge. I could determine the mass of the isotope by adding the number of protons to the number of neutrons.

Question 17: Given the name of an element and the number of neutrons, explain how you would find the mass of an isotope.

If I was given the name of an element, I could look at the periodic table and determine the atomic number. The atomic number is equal to the number of protons. Now knowing the number of protons I could add them to the number of neutrons to get the mass number.

Question 18: What are the differences between My Mix and Nature’s Mix?

In the Nature’s Mix the proportions are fixed and represent the relative abundances of the isotopes in nature. My Mix allows you to make the element have different proportions of each isotopes’ abundance.

Question 19: Give evidence to support or dispute: “In nature, the probability or chance of finding one isotope of an element is the same for all elements.” Use the simulation to write a paragraph for your argument.

I dispute the statement: “In nature, the chance of finding one isotope of an element is the same for all elements.” I will use three elements to illustrate my argument. The chances of finding one isotope of an element is different for most of the elements. It depends on the natural abundance of each isotope of each particular element. Lithium, Boron, and Oxygen all have different percent abundances.
